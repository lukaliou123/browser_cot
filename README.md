# 思维链 (ThoughtLoom) - 捕捉、连接、洞见你的网络思考

**Slogan:** 捕捉并可视化你的网络思考旅程，让灵感瞬间连接成系统知识。

[英文版 (English Version)](./README_EN.md) <!-- 预留英文版链接 -->

---

思维链 (ThoughtLoom) 是一款浏览器扩展，旨在帮助你轻松记录在网络上探索信息、学习新知时的思考路径。它不仅仅是书签，更是你思维的航迹图，将你浏览过的网页、产生的想法、以及它们之间的联系可视化地串联起来，并通过AI辅助你从信息流中提炼洞见，构建属于自己的知识体系。

## 核心解决的问题

你是否也曾有过这样的经历？

*   为了研究一个新课题或学习一项新技能，在网上打开了无数个标签页，信息零散，看过的东西很快就忘了。
*   好不容易找到一些有价值的资料，却难以将它们系统地组织起来，形成清晰的脉络。
*   想要回顾某个问题的探索过程，却发现记忆模糊，无法快速定位关键信息。

思维链 (ThoughtLoom) 正是为了解决这些痛点而生。

## 主要功能

*   ✨ **一键捕捉思维节点**：轻松记录当前网页，并可添加你的即时想法或笔记。
*   🧠 **思维链可视化**：以清晰的图谱形式展示你的思考路径，节点间的连接一目了然。
*   🔗 **多思维链管理**：
    *   自由创建、命名、切换不同的思维链，应对不同主题的探索。
    *   支持手动分割和每日自动创建新链，保持思考的条理性。
*   📝 **节点内容增强**：
    *   编辑和管理每个节点的笔记。
    *   （AI功能）自动为网页节点生成内容摘要，快速把握核心信息。
*   💡 **AI驱动的链总结**：
    *   （AI功能）一键为你当前探索的思维链生成一份包含洞察的结构化总结报告，梳理脉络，提炼精华。
    *   支持提供自定义指引，让AI总结更贴合你的需求。
*   🔍 **便捷的查看与回顾**：随时回顾和管理你的思维链。

## 如何改变你的信息处理方式？(用例展示)

想象一下，当你想学习一个全新的领域，比如"低代码开发"时，思维链如何帮助你：

---
**案例研究：小李学习"低代码"之旅**

**案例主角：** 小李，一个对"低代码/无代码开发"很感兴趣的上班族，他平时工作不直接接触编程，但希望了解这个趋势，看看是否能用来提升工作效率或搞点小副业。

**小李的痛点：**

*   网上关于"低代码"的信息很多，但很零散，有的是厂商广告，有的是技术深究，有的是成功案例，看得眼花缭乱。
*   他想知道"低代码到底是什么？"、"它和我现在的工作有什么关系？"、"如果想入门，有哪些靠谱的平台和学习资源？"
*   每次看完一些文章，关掉网页后，过几天就忘了看了啥，信息之间也串不起来。

**小李使用"思维链"应用的探索旅程：**

1.  **初步探索与疑惑（起点）：**
    *   小李在某个科技资讯网站看到一篇《"低代码"浪潮来袭，人人都是开发者？》的文章。他觉得标题很有吸引力，点进去看了看，对低代码有了初步印象，但还是很多问号。
    *   **思维链操作**：小李将这篇文章加入到他一个新的"低代码学习"思维链中，作为第一个节点。

2.  **了解基本概念与不同观点：**
    *   他在搜索引擎搜索"什么是低代码平台"，找到一篇知乎上的高赞回答，比较系统地解释了低代码的定义、优势和局限性。
    *   **思维链操作**：将该知乎回答加入思维链。
    *   他又看到一篇博客，标题是《别被忽悠了！低代码并非万能，这些场景请慎用》，提出了一些批判性观点。
    *   **思维链操作**：也加入思维链，帮助他更全面地思考。

3.  **寻找实际应用和工具：**
    *   小李想看看具体的低代码平台有哪些。他搜索"主流低代码平台推荐"，找到一个测评网站，对比了几个国内外流行的平台（比如国内的简道云、钉钉宜搭，国外的Airtable, Glide等）。
    *   **思维链操作**：将测评网站的对比页面加入思维链。
    *   他对其中一个平台A（比如Glide，因为它能快速把Google Sheets变成App，听起来很酷）产生了兴趣，于是访问了Glide的官网，看了它的功能介绍和用户案例。
    *   **思维链操作**：将Glide官网首页和某个案例页面加入思维链。

4.  **学习入门资源：**
    *   他想找找Glide的入门教程，于是在B站上搜到了一个UP主分享的"10分钟用Glide搭建一个信息收集App"的视频。
    *   **思维链操作**：将B站视频链接加入思维链。

5.  **形成链总结与感悟：**
    *   经过这一番探索，小李的"低代码学习"思维链已经有了6、7个节点，涵盖了新闻报道、概念解释、不同观点、工具测评、官网介绍、入门教程等。
    *   **思维链操作**：小李点击了"生成链总结"功能。
    *   **输出的链总结** (理想效果)：
        *   开头简明扼要地概括了低代码的核心理念（基于他看过的文章）。
        *   梳理了他探索过的信息脉络：从"什么是低代码"到"它有哪些争议"，再到"有哪些主流平台"以及"如何快速上手某个平台（Glide）"。
        *   可能还会提取出几个关键平台的名字和它们的主要特点。
        *   甚至能将教程视频的关键步骤或要点提示出来。
    *   **小李的感受**：
        *   "哇，太酷了！之前看了那么多网页，感觉脑子一团浆糊。现在这个总结把我的思路都理清了！"
        *   "它不仅记录了我看了什么，还帮我把这些信息串起来，形成了一个初步的知识框架。我现在对低代码是什么、能干什么、怎么学，心里有谱多了。"
        *   "这正是我想要的'捕捉并可视化我的网络思考旅程，让灵感瞬间连接成系统知识'的感觉！下次我要研究其他新东西，一定还用这个工具！"

**核心理念体现：**

这个案例清晰地展示了"思维链"应用如何帮助用户：
*   **捕捉网络思考旅程**：记录下小李在学习过程中浏览过的每一个关键信息点。
*   **可视化思考过程**：思维链本身就是一种可视化的路径。
*   **让灵感瞬间连接成系统知识**：通过链总结功能，将零散的信息点整合成结构化的认知，帮助小李从困惑到清晰，快速入门一个新领域。

## 安装与配置

### 1. 安装扩展

*   **方式一：从 Chrome 网上应用店安装 (尚未发布)**
    *   链接待提供。
*   **方式二：本地加载开发版本**
    1.  下载本项目代码到本地，或通过 `git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git` 克隆 (请替换为实际仓库地址)。
    2.  打开 Chrome 浏览器，进入扩展程序管理页面 (`chrome://extensions`)。
    3.  开启右上角的"开发者模式"。
    4.  点击"加载已解压的扩展程序"，选择本项目的根目录。
    5.  思维链扩展图标将出现在浏览器工具栏。

### 2. 配置 OpenAI API 密钥 (重要！)

本扩展的 AI 相关功能（如节点摘要、链总结）依赖 OpenAI API。你需要自行获取 API 密钥并进行配置。

1.  **获取 API 密钥**：访问 [OpenAI官方网站](https://openai.com/) 注册账户并获取你的 API 密钥。
2.  **配置到扩展中**：
    *   在项目代码中，找到 `js/config.js` 文件。
    *   打开该文件，你会看到如下内容：
        ```javascript
        // ... 其他配置 ...
        export const OPENAI_API_KEY = null;
        // ... 其他配置 ...
        ```
    *   将 `null` 替换为你的真实 OpenAI API 密钥（一个以 `sk-` 开头的字符串），例如：
        ```javascript
        export const OPENAI_API_KEY = 'sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx';
        ```
    *   保存 `js/config.js` 文件。
    *   **重要**：如果你的扩展是从本地加载的，修改配置后，请回到 `chrome://extensions` 页面，找到"思维链"扩展，点击刷新按钮（通常是一个环形箭头图标）使配置生效。
3.  **安全提示**：
    *   请妥善保管你的 API 密钥，不要泄露给他人。
    *   **切勿将包含你私人 API 密钥的 `js/config.js` 文件提交到任何公共代码仓库 (例如，如果你 fork 了本项目并公开了你的 fork)。**
    *   如果你不慎公开了密钥，请立即前往 OpenAI 官网禁用该密钥并生成新的。

## 快速上手

1.  点击浏览器工具栏的"思维链"扩展图标，打开主界面。
2.  当你浏览到一个你认为有价值的网页时，点击"捕捉当前页面"按钮，该页面就会作为一个新节点添加到你当前的思维链中。
3.  你可以为每个节点添加文字笔记，记录你的思考和感悟。
4.  当你完成一个主题的探索，或想对当前思维链进行梳理时，可以尝试"生成链总结"功能（确保已配置API密钥）。
5.  通过主界面的思维链列表，可以切换、管理你的不同思考路径。

## 技术栈 (简要)

*   **前端**: HTML, CSS, JavaScript (ES6 Modules)
*   **核心逻辑与数据管理**: `chrome.storage.local`, Service Worker
*   **可视化**: [D3.js](https://d3js.org/) (用于绘制思维导图)
*   **AI能力**: 通过 OpenAI API (gpt-3.5-turbo, gpt-4 等模型)
*   **文本提取**: [Readability.js](https://github.com/mozilla/readability) (用于提取网页正文)
